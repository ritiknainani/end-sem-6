## TYPE 4: MULTIPLE LINEAR REGRESSION + R² + F-TEST [10 Marks — 3/4 papers]

### Core Formulas:

**Model:** ŷ = a + b₁X₁ + b₂X₂

**Normal Equations:**
```
∑Y  = na   + b₁∑X₁  + b₂∑X₂           ...(I)
∑X₁Y = a∑X₁ + b₁∑X₁² + b₂∑X₁X₂       ...(II)
∑X₂Y = a∑X₂ + b₁∑X₁X₂ + b₂∑X₂²       ...(III)
```

**R² = SSR/SST = 1 − (SSE/SST)**

**F-test:**
```
F = [R²/k] / [(1−R²)/(n−k−1)]
k = number of predictors, n = number of observations
```

### ✅ Solved Example 7 — PYQ Paper 3 Q5a (Cost of Production)

**Q:** Data for 8 drugs — cost of production (Y), cost of ingredients (X₁), packaging cost (X₂). Fit ŷ = a + b₁X₁ + b₂X₂. Find R². Test significance (F = 5.786, α = 0.05).

| Sr | Y (₹) | X₁ (₹000) | X₂ (₹) |
|----|--------|-----------|---------|
| 1 | 100 | 17 | 19 |
| 2 | 79 | 50 | 54 |
| 3 | 100 | 90 | 36 |
| 4 | 129 | 30 | 36 |
| 5 | 158 | 15 | 16 |
| 6 | 106 | 20 | 25 |
| 7 | 58 | 20 | 24 |
| 8 | 78 | 50 | 8 |

**n = 8, k = 2**

**Step 1: Compute all required sums**

| Y | X₁ | X₂ | X₁Y | X₂Y | X₁² | X₂² | X₁X₂ | Y² |
|---|----|----|-----|-----|-----|-----|------|----|
| 100 | 17 | 19 | 1700 | 1900 | 289 | 361 | 323 | 10000 |
| 79 | 50 | 54 | 3950 | 4266 | 2500 | 2916 | 2700 | 6241 |
| 100 | 90 | 36 | 9000 | 3600 | 8100 | 1296 | 3240 | 10000 |
| 129 | 30 | 36 | 3870 | 4644 | 900 | 1296 | 1080 | 16641 |
| 158 | 15 | 16 | 2370 | 2528 | 225 | 256 | 240 | 24964 |
| 106 | 20 | 25 | 2120 | 2650 | 400 | 625 | 500 | 11236 |
| 58 | 20 | 24 | 1160 | 1392 | 400 | 576 | 480 | 3364 |
| 78 | 50 | 8 | 3900 | 624 | 2500 | 64 | 400 | 6084 |

**Sums:**
```
∑Y = 808        ∑X₁ = 292       ∑X₂ = 218
∑X₁Y = 28070    ∑X₂Y = 21604   ∑X₁² = 15314
∑X₂² = 7390     ∑X₁X₂ = 8963   ∑Y² = 88530
```

**Verification of sums:**
```
∑Y: 100+79+100+129+158+106+58+78 = 808 ✓
∑X₁: 17+50+90+30+15+20+20+50 = 292 ✓
∑X₂: 19+54+36+36+16+25+24+8 = 218 ✓
∑X₁Y: 1700+3950+9000+3870+2370+2120+1160+3900 = 28070 ✓
∑X₂Y: 1900+4266+3600+4644+2528+2650+1392+624 = 21604 ✓
∑X₁²: 289+2500+8100+900+225+400+400+2500 = 15314 ✓
∑X₂²: 361+2916+1296+1296+256+625+576+64 = 7390 ✓
∑X₁X₂: 323+2700+3240+1080+240+500+480+400 = 8963 ✓
∑Y²: 10000+6241+10000+16641+24964+11236+3364+6084 = 88530 ✓
```

**Step 2: Set up normal equations**
```
(I)   808  = 8a   + 292b₁  + 218b₂
(II)  28070 = 292a + 15314b₁ + 8963b₂
(III) 21604 = 218a + 8963b₁  + 7390b₂
```

**Step 3: Solve the system**

From (I): a = (808 − 292b₁ − 218b₂)/8 = 101 − 36.5b₁ − 27.25b₂

Substitute into (II):
```
28070 = 292(101 − 36.5b₁ − 27.25b₂) + 15314b₁ + 8963b₂
28070 = 29492 − 10658b₁ − 7957b₂ + 15314b₁ + 8963b₂
28070 − 29492 = 4656b₁ + 1006b₂
−1422 = 4656b₁ + 1006b₂      ...(IV)
```

Substitute into (III):
```
21604 = 218(101 − 36.5b₁ − 27.25b₂) + 8963b₁ + 7390b₂
21604 = 22018 − 7957b₁ − 5940.5b₂ + 8963b₁ + 7390b₂
21604 − 22018 = 1006b₁ + 1449.5b₂
−414 = 1006b₁ + 1449.5b₂      ...(V)
```

From (V): b₁ = (−414 − 1449.5b₂) / 1006

Substitute into (IV):
```
−1422 = 4656 × [(−414 − 1449.5b₂)/1006] + 1006b₂
−1422 = (4656 × −414)/1006 + (4656 × −1449.5)/1006 × b₂ + 1006b₂
−1422 = −1915.66 − 6706.49b₂ + 1006b₂
−1422 + 1915.66 = −5700.49b₂
493.66 = −5700.49b₂
b₂ = −0.0866
```

```
b₁ = (−414 − 1449.5(−0.0866)) / 1006
   = (−414 + 125.53) / 1006
   = −288.47 / 1006
   = −0.2867
```

```
a = 101 − 36.5(−0.2867) − 27.25(−0.0866)
  = 101 + 10.46 + 2.36
  = 113.82
```

**Regression Equation: ŷ = 113.82 − 0.287X₁ − 0.087X₂**

**Step 4: Calculate R²**
```
SST = ∑Y² − n·ȳ² = 88530 − 8(101)² = 88530 − 81608 = 6922

SSR = b₁(∑X₁Y − n·x̄₁·ȳ) + b₂(∑X₂Y − n·x̄₂·ȳ)

x̄₁ = 292/8 = 36.5,  x̄₂ = 218/8 = 27.25

∑X₁Y − n·x̄₁·ȳ = 28070 − 8(36.5)(101) = 28070 − 29492 = −1422
∑X₂Y − n·x̄₂·ȳ = 21604 − 8(27.25)(101) = 21604 − 22018 = −414

SSR = (−0.2867)(−1422) + (−0.0866)(−414)
    = 407.69 + 35.85
    = 443.54

R² = SSR/SST = 443.54/6922 = 0.0641
```

> Note: Low R² (6.4%) indicates ingredients and packaging alone don't explain production cost well.

**Step 5: F-test**
```
F_calc = [R²/k] / [(1−R²)/(n−k−1)]
       = [0.0641/2] / [(1−0.0641)/(8−2−1)]
       = 0.0321 / [0.9359/5]
       = 0.0321 / 0.1872
       = 0.171

F_table = 5.786 (given, α = 0.05, df₁ = 2, df₂ = 5)

Since F_calc (0.171) < F_table (5.786) → Fail to reject H₀
```
**Regression is NOT statistically significant.**

### ✅ Solved Example 8 — PYQ Paper 4 Q4a (Shipment Damage)

**Q:** Weights (X₁), Distance (X₂), Damage (Y) for 10 shipments. Fit ŷ = a + b₁X₁ + b₂X₂. Find R². Test (F = 9.55, α = 0.01).

| Ship | Y (₹00) | X₁ (kg) | X₂ (km) |
|------|---------|---------|---------|
| 1 | 12 | 17 | 10 |
| 2 | 15 | 15 | 6 |
| 3 | 14 | 15 | 10 |
| 4 | 19 | 10 | 21 |
| 5 | 8 | 15 | 5 |
| 6 | 16 | 15 | 13 |
| 7 | 15 | 11 | 9 |
| 8 | 25 | 6 | 25 |
| 9 | 10 | 15 | 15 |
| 10 | 11 | 7 | 8 |

**n = 10, k = 2**

**Step 1: Compute sums**
```
∑Y = 12+15+14+19+8+16+15+25+10+11 = 145
∑X₁ = 17+15+15+10+15+15+11+6+15+7 = 126
∑X₂ = 10+6+10+21+5+13+9+25+15+8 = 122

∑X₁Y = 204+225+210+190+120+240+165+150+150+77 = 1731
∑X₂Y = 120+90+140+399+40+208+135+625+150+88 = 1995
∑X₁² = 289+225+225+100+225+225+121+36+225+49 = 1720
∑X₂² = 100+36+100+441+25+169+81+625+225+64 = 1866
∑X₁X₂ = 170+90+150+210+75+195+99+150+225+56 = 1420
∑Y² = 144+225+196+361+64+256+225+625+100+121 = 2317
```

**Step 2: Normal equations**
```
ȳ = 145/10 = 14.5
x̄₁ = 126/10 = 12.6
x̄₂ = 122/10 = 12.2

(I)   145  = 10a  + 126b₁  + 122b₂
(II)  1731 = 126a + 1720b₁ + 1420b₂
(III) 1995 = 122a + 1420b₁ + 1866b₂
```

From (I): a = 14.5 − 12.6b₁ − 12.2b₂

Sub into (II):
```
1731 = 126(14.5 − 12.6b₁ − 12.2b₂) + 1720b₁ + 1420b₂
1731 = 1827 − 1587.6b₁ − 1537.2b₂ + 1720b₁ + 1420b₂
−96 = 132.4b₁ − 117.2b₂         ...(IV)
```

Sub into (III):
```
1995 = 122(14.5 − 12.6b₁ − 12.2b₂) + 1420b₁ + 1866b₂
1995 = 1769 − 1537.2b₁ − 1488.4b₂ + 1420b₁ + 1866b₂
226 = −117.2b₁ + 377.6b₂        ...(V)
```

From (IV): b₁ = (−96 + 117.2b₂)/132.4

Sub into (V):
```
226 = −117.2[(−96 + 117.2b₂)/132.4] + 377.6b₂
226 = (117.2 × 96)/132.4 − (117.2²/132.4)b₂ + 377.6b₂
226 = 84.92 − 103.74b₂ + 377.6b₂
226 − 84.92 = 273.86b₂
141.08 = 273.86b₂
b₂ = 0.5152
```

```
b₁ = (−96 + 117.2 × 0.5152) / 132.4
   = (−96 + 60.38) / 132.4
   = −35.62 / 132.4
   = −0.2690
```

```
a = 14.5 − 12.6(−0.269) − 12.2(0.5152)
  = 14.5 + 3.39 − 6.29
  = 11.60
```

**ŷ = 11.60 − 0.269X₁ + 0.515X₂**

**Step 3: R²**
```
SST = ∑Y² − nȳ² = 2317 − 10(14.5)² = 2317 − 2102.5 = 214.5

∑X₁Y − nx̄₁ȳ = 1731 − 10(12.6)(14.5) = 1731 − 1827 = −96
∑X₂Y − nx̄₂ȳ = 1995 − 10(12.2)(14.5) = 1995 − 1769 = 226

SSR = b₁(−96) + b₂(226)
    = (−0.269)(−96) + (0.515)(226)
    = 25.82 + 116.39
    = 142.21

R² = 142.21 / 214.5 = 0.663
```

**R² = 0.663 (66.3% of variation explained)**

**Step 4: F-test**
```
F_calc = [0.663/2] / [(1−0.663)/(10−2−1)]
       = 0.3315 / [0.337/7]
       = 0.3315 / 0.04814
       = 6.886

F_table = 9.55 (given, α = 0.01, df₁ = 2, df₂ = 7)

Since F_calc (6.886) < F_table (9.55) → Fail to reject H₀
```
**At α = 0.01, regression is NOT statistically significant.**

## TYPE 5: PARTIAL CORRELATION COEFFICIENT [5-10 Marks — 3/4 papers]

### Core Formulas:
```
              r₁₂ − r₁₃·r₂₃
r₁₂.₃ = ───────────────────────────
         √(1 − r₁₃²) · √(1 − r₂₃²)

              r₁₃ − r₁₂·r₂₃
r₁₃.₂ = ───────────────────────────
         √(1 − r₁₂²) · √(1 − r₂₃²)

Multiple correlation:
R₁.₂₃ = √[1 − (1 − r₁₂²)(1 − r₁₃.₂²)]
```

---

### ✅ Solved Example 9 — PYQ Paper 4 Q5a

**Q:** Given r₁₂ = 0.7, r₁₃ = 0.61, r₂₃ = 0.4. Compute r₁₂.₃, r₁₃.₂, and R₁.₂₃.

**Step 1: r₁₂.₃**
```
r₁₂.₃ = (r₁₂ − r₁₃·r₂₃) / [√(1 − r₁₃²) · √(1 − r₂₃²)]
       = (0.7 − 0.61 × 0.4) / [√(1 − 0.3721) · √(1 − 0.16)]
       = (0.7 − 0.244) / [√0.6279 · √0.84]
       = 0.456 / [0.7924 × 0.9165]
       = 0.456 / 0.7263
       = 0.628
```

**Step 2: r₁₃.₂**
```
r₁₃.₂ = (r₁₃ − r₁₂·r₂₃) / [√(1 − r₁₂²) · √(1 − r₂₃²)]
       = (0.61 − 0.7 × 0.4) / [√(1 − 0.49) · √(1 − 0.16)]
       = (0.61 − 0.28) / [√0.51 · √0.84]
       = 0.33 / [0.7141 × 0.9165]
       = 0.33 / 0.6544
       = 0.504
```

**Step 3: R₁.₂₃ (Multiple correlation coefficient)**
```
R₁.₂₃ = √[1 − (1 − r₁₂²)(1 − r₁₃.₂²)]
       = √[1 − (1 − 0.49)(1 − 0.2540)]
       = √[1 − (0.51)(0.746)]
       = √[1 − 0.3805]
       = √0.6195
       = 0.787
```

**Answers: r₁₂.₃ = 0.628, r₁₃.₂ = 0.504, R₁.₂₃ = 0.787**

### ✅ Solved Example 10 — Summer 2022 Q4C Style

**Q:** Given r₁₂ = 0.6, r₁₃ = 0.65, r₂₃ = 0.72. Calculate r₁₂.₃.

```
r₁₂.₃ = (r₁₂ − r₁₃·r₂₃) / [√(1 − r₁₃²) · √(1 − r₂₃²)]
       = (0.6 − 0.65 × 0.72) / [√(1 − 0.4225) · √(1 − 0.5184)]
       = (0.6 − 0.468) / [√0.5775 · √0.4816]
       = 0.132 / [0.7599 × 0.6940]
       = 0.132 / 0.5274
       = 0.2503
```

**Answer: r₁₂.₃ = 0.250**
